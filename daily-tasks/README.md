# Daily Tasks

This folder contains daily task-tracking markdown files for the RogueNet Live Services project.

## Naming Convention

Each daily file uses the format `YYYY-MM-DD.md` (e.g., `2026-05-05.md`).

## File Structure

Each daily file follows a standup/sprint style with epics, features, and work items:

```markdown
# YYYY-MM-DD — Daily Tasks

## Sprint Goal
<!-- One sentence describing what "done" looks like today. -->

## Today
- [ ] Work item or task

## In Progress
- [ ] ...

## Done
- [x] ...

## Blockers / Notes
<!-- Risks, open questions, or anything that needs follow-up. -->
```
